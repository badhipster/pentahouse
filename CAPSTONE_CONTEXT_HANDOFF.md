## Capstone Pick

**Real Estate Marketing & Conversion Intelligence Agent (Cohort 11 #2)**

Full-funnel multi-agent system that manages property listings, lead qualification, engagement, and site-visit conversion tracking.

### Agent Architecture (from official brief)

- **Listing Agent:** Syncs new properties from developer/aggregator feeds
- **Ad Agent:** Launches targeted campaigns on Meta/Google
- **Lead Agent:** Scores incoming inquiries using CRM + LLM
- **Nurture Agent:** Sends personalized WhatsApp updates to prospects
- **Conversion Agent:** Tracks site visits and bookings, closes the loop


### Tech Stack (expected)

- **Orchestration:** n8n (preferred by fellowship)
- **Ad Agent:** Meta Ads API, Google Ads API
- **Lead Agent:** CRM integration + LLM scoring (OpenAI GPT-4o)
- **Nurture Agent:** Twilio WhatsApp Business API
- **Conversion Agent:** Google Sheets + Booking APIs
- **Frontend:** Next.js or React (interactive UI layer required)
